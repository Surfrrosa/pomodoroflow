Version 1.0.0

September 27, 2025

Initial Release

  • Radically simple Pomodoro timer with zero configuration
  • Standard 25-minute focus / 5-minute break cycles
  • Gentle audio cues for session transitions
  • Haptic feedback for mindful notifications
  • Auto-progression through work and break sessions
  • Sessions persist even when app is closed
  • Local notifications to keep you on track
  • Fast Mode for rapid testing and QA
  • Dark mode interface designed for calm focus
  • Privacy-first: all data stored locally on device
  • No analytics, no tracking, fully offline capable

Design Philosophy

  • Built for neurodivergent minds and ADHD-friendly focus
  • Zero setup friction - open app and start focusing
  • Gentle, nervous-system friendly notifications
  • Minimalist UI that fades into the background
